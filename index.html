Sistema Experto Para Servicio de Red

Profesor: Horacio Bogarín

Alumnos: Valeria Villega, Cristian Couto, Diego Estrada

Objetivo del Proyecto

El objetivo de este proyecto es desarrollar un sistema experto interactivo que permita a los usuarios diagnosticar de forma autónoma problemas comunes en sus servicios de red domiciliarios (como internet, Wi-Fi, router, cableado, etc.). Este sistema simula el razonamiento de un técnico especializado, guiando al usuario paso a paso hasta identificar la causa probable del problema y sugerir soluciones concretas..
