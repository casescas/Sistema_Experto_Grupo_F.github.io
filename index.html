🎯 Objetivos

El sistema experto tiene como propósito principal diagnosticar problemas de conectividad en redes domésticas. Para ello, se basa en los siguientes pilares:

  • Definición de hechos observables: identificar síntomas como conexión lenta, ausencia total de internet y desconexiones intermitentes.
  • Creación de reglas de inferencia: establecer relaciones lógicas que permitan determinar la causa probable de cada problema.
  • Implementación de un endpoint con FastAPI: ofrecer una interfaz eficiente para realizar pruebas simultáneas en múltiples dispositivos, garantizando rapidez y escalabilidad.

Este enfoque busca optimizar la detección de fallas y mejorar la experiencia del usuario mediante un sistema automatizado, confiable y fácil de integrar.
