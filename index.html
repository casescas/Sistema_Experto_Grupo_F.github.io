🔍 Introducción

Este sistema experto permite diagnosticar fallas comunes en redes domésticas mediante reglas basadas en síntomas observables.

🧠 Arquitectura

El sistema utiliza una base de hechos, reglas encadenadas y un motor de inferencia para llegar a conclusiones explicables.

📦 Casos de Uso

  • Falla de conexión Wi-Fi
  • Problemas con el router
  • Interferencia de señal
  • Configuración incorrecta de IP

📊 Resultados

El sistema ha demostrado una alta precisión en la identificación de fallas y ofrece recomendaciones claras para su resolución.
